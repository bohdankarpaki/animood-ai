# ⛩️ AniMood AI

**AniMood AI** — це інтелектуальний вебзастосунок, який аналізує поточний настрій користувача та за допомогою великих мовних моделей (LLM) генерує ідеальну рекомендацію аніме з детальним атмосферним описом.

Проєкт розроблено в рамках індивідуального завдання з дисципліни «Вебпрограмування» (**Національний університет «Львівська політехніка»**).

🔗 **[Live Demo (Vercel)](https://animood-ai.vercel.app/)**

---

## 🚀 Головні функції

- **🧠 Каскадна система ШІ**: Використання інтелектуальної ротації моделей Google Gemini (**3.1 Pro**, **3.0 Flash**, **2.5 Flash/Pro**, **2.0 Flash-Lite**) для забезпечення стабільної роботи навіть при перевантаженні API.
- **⚡ Потокова генерація (Streaming)**: Відповідь від ШІ з'являється на екрані частинами в реальному часі (як у ChatGPT).
- **🎭 Динамічний UI**: Автоматична зміна колірної схеми та атмосфери застосунку залежно від введеного настрою.
- **🔐 Авторизація**: Інтегрований Google Login через Firebase Auth.
- **🗂️ Сховище (Обране та Історія)**: 
  - Збереження улюблених тайтлів у власну колекцію.
  - Автоматичне збереження історії генерацій з можливістю перегляду без витрати лімітів.
- **🛡️ Контроль лімітів (Usage Quotas)**: 3 запити на день для гостей, 15 для авторизованих користувачів.
- **👑 Supreme Architect Mode**: Спеціальний режим для адміністратора з безлімітним доступом та панеллю аналітики.
- **🖼️ Динамічні обкладинки**: Автоматичне підтягування постерів та рейтингів через Jikan API (MyAnimeList).

---

## 🛠️ Стек технологій

- **Фронтенд**: Next.js 14+ (App Router), React, Tailwind CSS
- **Бекенд / API**: Next.js Route Handlers, Google Generative AI SDK
- **База даних**: Firebase Firestore
- **Автентифікація**: Firebase Auth
- **Деплой**: Vercel
