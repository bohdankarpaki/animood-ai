# ⛩️ AniMood AI

**AniMood AI** — це інтелектуальна екосистема для персоналізованого підбору аніме. Застосунок аналізує настрій користувача за допомогою великих мовних моделей (LLM) та генерує рекомендації з глибоким атмосферним описом, забезпечуючи взаємодію в реальному часі.

Проєкт розроблено як індивідуальне завдання з дисципліни «Вебпрограмування» (**Національний університет «Львівська політехніка»**).

🔗 **[Live Demo (Vercel)](https://animood-ai.vercel.app/)**

---

## 🚀 Основні технічні особливості

- **🧠 Каскадна система LLM (Multi-Model Fallback)**: Інтелектуальна ротація API-ключів та каскадне перемикання між моделями (**Gemini 3.1 Pro, 3.0 Flash, 2.5 Flash/Pro, 2.0 Flash-Lite**). Це гарантує стабільність генерації навіть за умови вичерпання квот одного з ключів.
- **⚡ Потокова генерація (AI Streaming)**: Використання `ReadableStream` для відображення відповіді ШІ посимвольно в реальному часі.
- **🎭 Context-Aware UI**: Динамічна зміна кольорової схеми (атмосфери) всього інтерфейсу на основі семантичного аналізу введеного тексту.
- **📡 Real-time Synchronization (Firestore onSnapshot)**: Миттєве оновлення лімітів та сповіщень на сайті без перезавантаження сторінки. Як тільки адміністратор приймає рішення, користувач бачить Toast-повідомлення через WebSocket-з'єднання з Firebase.
- **🛡️ Безпека та контроль квот**: 
  - Динамічні ліміти: 3 запити для гостей, 15+ для авторизованих користувачів.
  - **One-time Request Tokens**: Система схвалення лімітів захищена унікальними `requestId`, що робить посилання в адмін-листах одноразовими та безпечними.
- **👑 Supreme Architect Dashboard**: Ексклюзивна панель для адміністратора з реальною аналітикою:
  - Підрахунок унікальних користувачів та загальної кількості генерацій прямо з сервера Firebase.
  - **Динамічний аналіз трендів**: Автоматичне вирахування найпопулярніших настроїв серед користувачів на основі аналізу історії запитів.
- **🖼️ Інтеграція з Jikan API**: Автоматичний пошук метаданих (обкладинки, рейтинги, посилання) на MyAnimeList для кожної генерації.

## 🛠️ Технологічний стек

- **Framework**: Next.js 14+ (App Router, Turbopack)
- **Frontend**: React, Tailwind CSS, Framer Motion, Lucide React
- **Database/Auth**: Firebase Firestore & Firebase Auth
- **AI Engine**: Google Generative AI SDK (Gemini API)
- **Mail System**: Nodemailer (SMTP integration)
