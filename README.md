# ⛩️ AniMood AI

**AniMood AI** — це інтелектуальний вебзастосунок, який аналізує поточний настрій користувача та за допомогою великих мовних моделей (LLM) генерує ідеальну рекомендацію аніме з детальним атмосферним описом.

Проєкт розроблено в рамках індивідуального завдання з дисципліни «Вебпрограмування» (Національний університет «Львівська політехніка»).

🔗 **[Live Demo (Vercel)](https://animood-ai.vercel.app/)**

---

## 🚀 Головні функції

- **🧠 ШІ-рекомендації**: Використання Google Gemini (2.0 Flash-Lite / 1.5 Flash) для аналізу настрою та генерації атмосферних описів.
- **⚡ Потокова генерація (Streaming)**: Відповідь від ШІ з'являється на екрані частинами в реальному часі (як у ChatGPT), без необхідності чекати повного завершення запиту.
- **🔐 Авторизація**: Інтегрований Google Login через Firebase Auth.
- **🗂️ Сховище (Обране та Історія)**: 
  - Збереження улюблених тайтлів у власну колекцію.
  - Автоматичне збереження історії генерацій з можливістю миттєвого перегляду без витрати лімітів.
- **🛡️ Контроль лімітів (Usage Quotas)**: Захист API від зловживань. Неавторизовані користувачі мають 3 запити на день, авторизовані — 15.
- **🖼️ Динамічні обкладинки**: Автоматичне підтягування постерів, рейтингів та посилань через Jikan API (MyAnimeList).

---

## 🛠️ Стек технологій

- **Фронтенд**: Next.js 14+ (App Router), React, Tailwind CSS
- **Бекенд / API**: Next.js Route Handlers, Google Generative AI SDK
- **База даних та Авторизація**: Firebase (Firestore, Auth)
- **Зовнішні API**: Jikan API (v4)
- **Деплой та Аналітика**: Vercel (Speed Insights, Web Analytics)
